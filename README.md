# GTA Parcel Delivery - Marketing & Lead Generation System

## Project Overview
A complete marketing and lead generation system for a local parcel delivery service targeting businesses in the Greater Toronto Area (GTA).

## Features
- ✅ Professional 5-page website (Home, Services, About, Contact, Thank You)
- ✅ Lead capture forms with validation
- ✅ Complete CRM dashboard for lead management
- ✅ Follow-up scheduling and tracking
- ✅ Communication logging
- ✅ Quote management system
- ✅ Excel import/export functionality
- ✅ Mobile responsive design
- ✅ SEO optimized for local keywords
- ✅ Google Business Profile integration ready

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5, Django Templates
- **Backend**: Python 3.9+, Django 4.2
- **Database**: SQLite (development) / PostgreSQL (production)
- **Additional**: Celery, Redis, Twilio API
